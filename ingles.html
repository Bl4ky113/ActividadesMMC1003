Diferentes Presentes o ¿diferentes Realidades?

El presente simple or Present Simple, es el que se usa para informar sobre un presente generalmente indefinido. Para usar la afirmación escribimos el verbo normal o con s, es, dependiendo de la persona junto a la persona y sus complementos. Para usar la negación usamos un negador not n't al final del verbo, junto todo los demás. Y para una pregunta, ponemos Do o Does, depende de la persona, de primeras o antes del WH Question si hay alguna, persona, verbo sencillo y el complemento.

El presente continuo or Present Continuous, es el que se usa para indicar sobre una situación que esta ocurriendo. Para usar la afirmación se usa la persona, el V2B, el verbo con ing y otros cambios dependiendo del verbo. Y sus complementos. Para negación, se niega el V2B, se deja el verbo con ing quieto y los complementos. Para una pregunta, se usa el V2B de primeras, o antes de una WH Question, la persona, el verbo con el ing y los complementos.

El presente perfecto or Present Perfect, es el que se usa para indicar de una situación que inicio en el pasado y que continuan pasando o que van a seguir pasando. Para usar la afirmación se usa la persona, el have o has, depende de la persona, verbo y complemento. Para usar la negación se niega el have o el has y se deja quieto lo demás. Para usar la pregunta se usa el have o el has, antes de la WH Question si es necesario, la persona, el verbo y sus complementos.

El presente perfecto continuo or Present Perfect Continuous, es el que se usa indicar de una situación que inicio en el pasado y que aún continua en el presente. Para usar la afirmación, se usa la persona, el have | has, el verbo con ing y sus complementos. Para usar la negación se niega el have | has y todo lo demás quieto. Para usar la pregunta se usa de primeras el WH Question si hay, despues el have | has, persona, verbo con ing y sus complementos.

Diferentes Pasados o ¿Errores Que Pudiste Evitar?

El pasado simple or Past Simple, es el que se utiliza para decir actividades o rutinas que toman lugar en el pasado. Pero para esto los verbos se dividen en 2 grupos. El Verbo en Pasado Regular y el Verbo en Pasado Iregular. El Verbo en Pasado Regular, son los verbos que terminan en ed y el Verbo en Pasado Irregular, son los verbos que terminan de una forma espacial.
Para usar la afirmación, se usa el sujeto, el verbo Regular o Irregular y sus complementos. Para usar la negación, se usa el sujeto, did (para todos los sujetos), negación not n't, verbo normal y sus complementos. Para usar la pregunta, se usa el did o primero la WH Question, el sujeto, el verbo y sus complementos.

El pasado continuo or Past Continuous, es el que se utiliza para acciones que estaban pasando en un momento especifico en el pasado. Este no usa los verbos irregular o regular, solo usa el V2B en pasado y los verbos con ing.
Para usar la afirmación, se usa el sujeto, el was para personas singulares y el were para personas plurales. No confundir con el we're, el verbo con ing y sus complementos. Para usar la negación, solo se niega el was o were. Para usar la pregunta, se usa el was o were en el inicio o despues de una WH Question y ya.

El pasado perfecto or Past Perfect, es el que se utiliza para indicar alguna situación que ocurrió antes de otra situación. Este pasado se usa tambien los verbos regualres o los verbos irregulares, y el verbo had. Para usar la afirmación, se usa el sujeto, el had, el verbo regular o verbo irregular y sus complementos. Para usar la negación, se usa el sujeto, el had negadado, el verbo regular o verbo irregualar y sus complementos. Para usar la pregunta, se usa el had o despues del WH Question, el sujeto, el verbo regular o verbo irregular, y sus complementos.

El pasado perfecto continuo or Past Perfect Continuous, es el que se utiliza para indicar sobre una situación que ocurrio hace poco desde ahora. Se usa el had junto a el been y los verbos con ing. Para usar la afirmación, se usa el sujeto, el had been, el verbo con ing y sus complementos. Para usar la negación, se usa el sujeto, el had negado junto a el been, el verbo con ing y sus complementos. Para usar la pregunta, se usa el had o despues del WH Question, el sujeto, el been, el verbo con ing y sus complementos.

Tipos de Personas

Se dividen en 3 tipos:

  • 1ra persona: I
  • 2da persona: We, They, You
  • 3ra persona: She, He, It

Simple Present

Se puede dividir en 3 partes, Afirmativo, Negativo y pregunta, tienen muy pocas diferencias, pero importantes. En la 1ra y 2da persona se debe hacer la oración sencilla, sin ningun cambio en el verbo o en el complemento. PERO en la 3ra persona se debe agregarle una "S" a el verbo

  • I Like Mathematics
  • We Like Mathematics
  • She Likes Mathematics

El verbo have en la 3ra Persona se debe quitar el "ve" y agregar "s".

El uso de la "S" en el Presente Simple

Primero debemos aclarar que la S solo se va a aplicar a la primera palabra de los verbos que sean de un noun de tercera persona singular o third person singular. Estos son:

  • She
  • He
  • It

Listo! ahora para poder añadir la S y otros complementos a los verbos, primero de deben divir estos en 4 grupos.

Verbos Regulares

Son los verbos que se van a diferenciar de todos los siguientes verbos, estos son los más regulares. En estos verbos simplemente debemos agregar el sufijo "-s" a el final de estos. Ejemplo:

  • fights
  • wonders
  • thinks

Verbos terminados en...

Son los verbos que terminan en:

  • s
  • z
  • x
  • ch
  • sh

A estos verbos se les agrega el sufijo "-es" a el final de estos. Ejemplos:

  • Kisses
  • Catches
  • Wishes

Verbos terminados en "Y"

Son los verbos que terminan en "Y" y antes de esta tienen una consonante, no toca confundir estos verbos con los verbos que terminan en las anteriores denominaciones. (s, z, x, ch, sh). A estos verbos se les quita la "Y" y esta se cambia por el sufijo "ies". Ejemplo:

  • Flies
  • Dries
  • Fries

Verbos Especiales

Estos verbos con especiales, pero son la menor minoria en todos los verbos, solo teniendo 3 verbos en este grupo. Estos verbos son:

  • Have
  • Go
  • Do

Que estos se les agregaría un sufijo diferente a los que deberían tener. Ejemplo:

  • Has
  • Goes
  • Does

Present Perfect

Lunes | 04/05/2021

Se usa para decir acciones que han pasado en el pasado, es decir: he comdio, hecho, hablado, etc.

Como usarlo

El Presente Perfecto se usa el Have y Has (3ra Persona), este cambia su significado a : he, hemos, han o ha. Para formar la sentencia usando Present Perfect, se debe usar:
Persona + Have/Hav + Verbo en Pasado Participo + Complementos.

I Have eaten Bread loafs. He Has eaten Bread Bars.

En Negativo

Simplemente le agregamos una negación a el have o has, siendo:
Persona + Have/Has + Not + Verbo en Pasado Participo + Complementos

Traducciones Sencillas

  • She has frozen water
  • We have found Petrol
  • He has cut the grass in our garden
  • He cuts the grass in our garden
