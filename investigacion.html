Con tus palabras describe el significado y la relación entre ciencia, técnica y tecnología.

    Ciencia

  • Conjunto de conocimientos, experimentos, teorias, leyes que son verdaderos y que ayudan a comprender mucho más el porqué y el como de las cosas, desde un punto técnico, biologíco, químico, físico, etc. Ejemplo, él saber que hace el fuego a nivel molecular.

    • Técnica

  • Conjunto de acciones que se tienen que hacer para obtener siempre el mismo o aproximado resultado, se puede mejorar con diferentes formas de hacer las acciones o con herramientas, tecnología, entre otros. Ejemplo, él usar el fuego para calentar agua y poder cocinar.

    • Tecnología

  • Conjunto de avances, físicos, electrónicos, químicos, cientificos aplicados a diferentes herramientas para obtener un mejor rendimiento frente a diferentes acciones o técnicas. Ejemplo, él usar el fuego y agua para hacer una maquina de vapor.
