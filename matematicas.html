Como obtener el valor de un Angulo en GeoGebra

En la herramienta de Geometría, elegimos la herramienta de un Segmento, unimos dos Segmentos en el plano. Depues buscamos en el apartado de Mediciones, la herramienta de Medir Angulos, elegimos los 3 vertices de cada Segmento. Y GeoGebra nos va a mostrar el resultado, si el resultado esta muy oculto o no visible, lo podemos mover con la herramienta de mover. O podemos ver la lista de movimientos y resultados, hechos en la calculadora de GeoGebra.

Nombrar un Angulo

Para nombrar un Angulo, debemos primero mirar si el angulo, es un angulo sencillo, es decir solo tiene 2 segmentos. En este caso el angulo tendrá el nombre del vertice donde se origina el angulo. Por ejemplo:

Este tiene un angulo sencillo, en este caso es B, es el angulo, por lo tanto B es el nombre de este angulo.

Para obtener el nombre de un angulo más complejo, debemos ver en cuantos angulos se divide el vertice, minimo deben ser más de 1 angulo por vertice, para hacer un angulo más complejo. Miremos este ejemplo:

Vamos a obtener el nombre del angulo de A, pero toca tener cuidado, A, tiene 4 Angulos. El primer angulo sera el angulo entre los segmentos A, B y E. Vamos a tomar B, como primer segmento, A, como el vertice donde esta el angulo y E, como el otro segmento.
El nombre de este angulo sería : BAE. O EAB, si tomamos el segmento E primero y despues el B.
Seguimos aplicanto esta tecnica, en los angulos restantes y obtendremos todos los nombres de esos angulos.

  • BAE - EAB
  • CAE - EAC
  • BAC - CAB
  • BAC - CAB (Angulo Externo)

Trucos con GeoGebra

02/18/2021

El segmento donde inicia la medición o 0°, se llama segmento inicial, y el otro segmento, se llama segmento final.

Cuando el angulo inicia en el sentido contrarío de un reloj, este angulo sera positivo. Cuando el angulo inicia en el sentido correcto de un reloj, este angulo es negativo. Se puede usar la herramienta Angle with given size, para obtener diferentes angulos en diferentes sentidos del reloj. Tambíen se puede entrar a la configuración de los Angulos para usar diferentes styles para el Angulo, y poder hacer otras cosas, como usar los angulos negativos dentro de GeoGebra.
