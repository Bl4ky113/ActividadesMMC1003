Para nombrar un Angulo, debemos primero mirar si el angulo, es un angulo sencillo, es decir solo tiene 2 segmentos. En este caso el angulo tendrá el nombre del vertice donde se origina el angulo. Por ejemplo:

Este tiene un angulo sencillo, en este caso es B, es el angulo, por lo tanto B es el nombre de este angulo.

Para obtener el nombre de un angulo más complejo, debemos ver en cuantos angulos se divide el vertice, minimo deben ser más de 1 angulo por vertice, para hacer un angulo más complejo. Miremos este ejemplo:

Vamos a obtener el nombre del angulo de A, pero toca tener cuidado, A, tiene 4 Angulos. El primer angulo sera el angulo entre los segmentos A, B y E. Vamos a tomar B, como primer segmento, A, como el vertice donde esta el angulo y E, como el otro segmento.
El nombre de este angulo sería : BAE. O EAB, si tomamos el segmento E primero y despues el B.

El segmento donde inicia la medición o 0°, se llama segmento inicial, y el otro segmento, se llama segmento final.

Cuando el angulo inicia en el sentido contrarío de un reloj, este angulo sera positivo. Cuando el angulo inicia en el sentido correcto de un reloj, este angulo es negativo. Sistema Decimal

En la suma del sistema decimal se cuenta dividen los números en unidades, decenas, centenas... Cada 10 de una se le suma 1 a la siguiente tipo de undidad, dandole un valor de 1 undidad por 10.

Bases de las Unidades
Base 10 102 101 100
Unidad Centenas Decimas Unidades

Para cambiar de base 10, debemos hacer grupos de la base que vallamos a hacer, como el sistema binario, hexadecimal o de 60 (Minutos y Horas). Entonces cada 60, 1 o 15 números aumentamos una unidad, estas pueden cambiar de nombre dependiendo del sistema, pero siempre van a existir en cualquiera.

Sistema Sexadecimal

El sistema sexadeciamal se divide en 360 pedazos, estos se llaman Grados en el sistema de Grados, existen otros sistemas como el Radiales en el que se usan 2 Pí.

Converciónes

Pí siempre va a ser 180° , cosa que nos va a ayudar mucho en la converción.

Grados a Radial

Para convertir de Grados a Radiales, debemos multiplicar el grado por Pí y a esto lo dividimos por 180, para hacer más efectiva y sencilla la converción, podemos simplificar el 180 junto a el grado.
